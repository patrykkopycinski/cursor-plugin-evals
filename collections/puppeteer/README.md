# Puppeteer MCP Server Collection

Integration tests for `@modelcontextprotocol/server-puppeteer`.

Covers: `puppeteer_navigate` with URL and title verification,
`puppeteer_screenshot` for full page and element captures,
`puppeteer_evaluate` for JavaScript execution and math,
`puppeteer_click`, `puppeteer_fill`, `puppeteer_hover`,
and error cases for nonexistent selectors.
