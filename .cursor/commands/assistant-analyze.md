Use the **report-analyzer** skill to analyze recent evaluation results:
1. Find the latest eval run results
2. Cluster failures by root cause (wrong tool, wrong args, hallucination, etc.)
3. Check for regressions against the baseline fingerprint
4. Identify flaky tests (inconsistent results across repetitions)
5. Analyze cost optimization opportunities
6. Check if CI thresholds are well-calibrated
7. Cross-reference failures with coverage gaps

Present an actionable analysis report with prioritized suggestions.
